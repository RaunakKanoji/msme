# Testing guide

Run `pnpm type-check`, `pnpm lint`, `pnpm test`, and `pnpm build`. Unit coverage includes consent mapping, state handling, decimal parsing, normalization, deduplication, minimum metric requirements, and feature generation. Sandbox end-to-end testing requires Bridge callbacks, approved test identities, Setu-hosted consent, FI notifications, fetch, revocation, and duplicate-event checks.
